# page-creator

create static page by next.js

问题：layout跳转不了

### Build

```bash
  $ npm run build
```

### Export

export前先要启动服务

```bash
  $ npm run export
```


### 启动服务

```bash
  $ npm run server
```


### 测试连接

```js
  http://localhost:3001
  http://localhost:3001/page1
  http://localhost:3001/page2
```
